# Ladder Toss: Backyard Casual

**Hyper-casual backyard game for Tier-1 markets (USA, Canada, UK, Germany, Australia)**  
2026 | AfyanLevon Studio

## Game Description
Classic Ladder Toss (Ladder Golf / Bolas) turned into addictive hyper-casual physics game.  
Swipe to throw bolas → score 21 points first!

## Tech Stack
- Unity 2026 LTS
- Git LFS
- AdMob + Unity Ads
- Google Play Games Services

## Folder Structure
- `_Core/` → reusable scripts (մեր գործարանի համար)
- `_Project/` → այս խաղի հատուկ ֆայլեր

## Workflow
1. `develop` branch-ում աշխատել
2. Feature branches → PR → merge to develop
3. Production build-եր միայն `main`-ում
